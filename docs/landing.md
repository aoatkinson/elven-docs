# Welcome to Elven

Elven is a modern Virtual Tabletop (VTT) platform providing a flexible, self-hosted environment to run and customize your tabletop RPG sessions. Whether you're just playing or building tools around the system, this documentation covers everything you need.

<TipBox>
**Want to play now? Try the live demo**  at [elvenvtt.com](https://elvenvtt.com)
</TipBox>
